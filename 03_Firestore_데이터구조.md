## Firebase Storage 구조

### 제품 이미지 경로
```
products/
  ├── 11011.jpg
  ├── 11012.jpg
  ├── 12222.jpg
  └── ...
```

이미지 탐색 순서:
1. `products/{코드}.jpg`
2. `products/{코드}.png`
3. `products/{코드}.JPG`
4. `products/{코드}.jpeg`
